# ZenHP（企業サイト風Webサイト）

ZenHP は、企業サイトを想定して制作した **静的Webサイト（複数ページ構成）** です。  
ナビゲーションの分かりやすさ・読みやすいレイアウト・シンプルなデザインを意識して作成しました。

---

🔗 **デモサイトはこちら**  
https://alecmcclendon.github.io/ZenHP/

---

## 🛠 使用技術

- フロントエンド：HTML / CSS（必要に応じて JavaScript）
- デプロイ：GitHub Pages

---

## 🔍 主なポイント（こだわり）

- 複数ページ構成（トップ / Company / History / Contact）
- シンプルで見やすいレイアウト・導線
- 画像などの静的ファイルを `assets/` で管理
- GitHub Pages で公開し、URL共有できる形に整備

---

## 📄 ページ一覧

- `index.html`：トップページ
- `company.html`：会社情報ページ
- `history.html`：沿革ページ
- `contact.html`：お問い合わせページ
- `assets/`：画像・素材など

---

## ▶ 使い方（ローカルで開く）

1. このリポジトリをクローン / ダウンロード
2. `index.html` をブラウザで開く
