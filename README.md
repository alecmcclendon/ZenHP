# ZenHP（企業サイト風Webサイト）

ZenHP は、**アルバイト先の会社向けに制作したコーポレートサイト（静的Webサイト）** です。  
PC / スマートフォンの両方で見やすいように **レスポンシブ対応** を行い、シンプルで分かりやすい導線を意識して作成しました。

---

🔗 [デモサイトはこちら](https://alecmcclendon.github.io/ZenHP/)

---

## 🛠 使用技術

- フロントエンド：HTML / CSS / JavaScript
- デプロイ：GitHub Pages

---

## 🔍 主なポイント

- **企業サイトを想定した複数ページ構成**
- **レスポンシブ対応（PC / Mobile）**
- 迷わないナビゲーションと読みやすいレイアウト
- 画像などの静的ファイルを `assets/` で管理

---

## 📄 ページ一覧

- `index.html`：トップページ
- `company.html`：会社情報ページ
- `history.html`：沿革ページ
- `contact.html`：お問い合わせページ
- `assets/`：画像・素材など
